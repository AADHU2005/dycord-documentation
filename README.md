> Dycord is a multipurpose bot made with [discord.js](https://discord.js.org) library by [Aadhu](https://discord.com/users/815480311285547079).

* Welcome Module

  * Deletes After Seconds
* Chatbot
* Utility commands
* Giveaway commands
